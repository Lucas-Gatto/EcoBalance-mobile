COMO INICIAR PROJETO (Mobile)
- Configure um emulador e deixe-o ativo
- No terminal do projeto: cd ecobalance
- Ainda no terminal: npm i
- Ainda no terminal: npm run android

COMO INICIAR BACKEND
- Veja se o .env tem conteúdo, se não, entre em contato com o Ricardo
- Em um outro terminal: cd backend
- npm i
- npm run dev
